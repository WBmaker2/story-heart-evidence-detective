# 릴리스 준비 상태

## 자동 게이트

`npm run verify`는 다음 항목을 한 번에 확인합니다.

- TypeScript와 ESLint
- 코드·스타일·스크립트·SVG 파일 500줄 미만
- 이야기 은행, 본문 앵커, 데이터 참조 무결성
- 앱과 워커 코드의 브라우저 저장소·서버 쿠키·외부 런타임 요청 부재
- 학생 화면의 점수·정답·오답 표현 부재
- 스타터 UI·메타데이터·D1/Drizzle 예제와 `codex-preview` 부재
- 키보드 포커스, 320px 최소 폭, 모션 감소, 강제 색상, 터치 영역 CSS 계약
- 단위 테스트, 프로덕션 빌드, 서버 렌더링 한국어 제품 HTML

문자열·소스 계약 검사는 실제 브라우저나 보조공학의 동작을 대신하지 않습니다.

## 수동 릴리스 게이트 — 대기 중

- [ ] 초등 저학년 지도 경험자 검토자 2인의 독립 콘텐츠 검수
- [ ] 검토 의견 반영 뒤 콘텐츠 버전·앵커·업데이트 내역 재확인
- [ ] 실제 태블릿에서 320px 이상 화면, 세로·가로 흐름, 48px 터치 영역 확인
- [ ] 실제 기기의 VoiceOver 또는 TalkBack으로 제목, 라디오, 체크박스, 초점 이동, 피드백, 대화상자 확인

위 네 항목은 현재 완료로 표시하지 않습니다. 완료 증거가 기록되기 전에는 사람 검수와 실제 기기 접근성 검증이 끝났다고 안내해서는 안 됩니다.
