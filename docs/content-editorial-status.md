# 콘텐츠 편집 상태

콘텐츠 버전 1.0.0의 이야기 7편과 복수 해석 지도는 앱 연결을 위한 편집 초안입니다. 자동 검증은 본문 앵커, 데이터 참조, 고정 사건 순서, 후보·근거·해석 구조의 무결성만 확인합니다. 문학적 타당성, 저학년 어휘 적합성, 빠진 해석이나 억지 조합까지 자동으로 보증하지 않습니다.

## 미완료 수동 검증

- 초등 저학년 지도 경험자 2명이 서로 독립적으로 가능한 마음과 정확한 근거 조합을 검토해야 합니다.
- 검토 결과에는 원고 버전, 검토자 역할, 검토일, 누락·편견·어휘 적합성·조합 수정 기록이 필요합니다.
- 콘텐츠가 바뀌면 두 사람의 재검토와 본문 앵커 자동 검증을 모두 다시 수행해야 합니다.
- 실제 태블릿과 실제 기기의 VoiceOver 또는 TalkBack 검증은 별도 출시 전 단계이며 아직 완료되지 않았습니다.
